About

dbbackup is a simple Perl script I use to backup databases I have on various hosting services. I run it a scheduled job to create a local database dump which I can then get off the hosting service and onto my backup system.

The script uses a configuration file for essential infomration (like where to put the backups). In its simplest form it queries the database host for the list of database instances and makes a backup of each individually.
